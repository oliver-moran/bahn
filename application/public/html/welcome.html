Welcome to bahn

A ready-for-road HTML5 application stack combining Bootstrap, AngularJS, H5BP, and Node.js (BAHN).

bahn is a HTML5 application stack combining the best of the modern web:

  • Bootstrap
  • AngularJS
  • HTML5 Boilerplate
  • Node.js

With Node.js, we've bundled the Express application server, a NoSQL database, NeDB, and real-time client-server communication with WebSockets and Socket.io. For high-traffic installations, bahn lets you switch to MongoDB with the flick of a switch (using the mongojs API).
