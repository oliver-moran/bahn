Example Application

This classic TODO example puts bahn through its paces, using Bootstrap, AngularJS, H5BP, and Node.js.

Bootstrap, AngularJS and H5BP handle the data and presentation on the client side. The bundled NoSQL database, NeDB, or MongoDB, stores tasks on the server. Express, serves up the list of tasks when the page is loads over HTTP using REST (see /api/todo/tasks). Socket.io broadcasts changes to multiple clients instantly over WebSockets.
