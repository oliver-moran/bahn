Welcome to bahn

Ready-for-road application stack for HTML5 with Bootstrap, AngularJS, H5BP, and Node.js.

bahn is a HTML5 application stack combining the best of modern web:

  • Bootstrap
  • AngularJS
  • HTML5 Boilerplate
  • Node.js

With Node, we've bundled a NoSQL database, NeDB, and real-time client-server communication with Socket.io and JSON-RPC 2.0. For high-traffic installations, bahn lets you switch to MongoDB with the flick of a switch.
