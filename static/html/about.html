Welcome to bahn {{version}}

Ready-for-road application stack for HTML5 with Bootstrap, AngularJS, H5BP, and NodeJS.

bahn is a HTML5 application stack built combining:

  • UI framework (Bootstrap)
  • MVVM framework (AngularJS)
  • No SQL database (nedb)
  • Web socket server (Socket.io)
  • HTML scaffolding (HTML5 Boilerplate)
