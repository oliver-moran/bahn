What's inside?

bahn pulls together the best modern frameworks and technologies in an agnostic package that let's you choose what you need for your project.

HTML5 application framework

bahn pulls together three of the moderns web's most popular frameworks for HTML5 development: Bootstrap, AngularJS, and the HTML5 Boilerplate.

HTTP application server

Express is the top HTTP application server for Node.js.

NoSQL databse

bahn offers two options for a NoSQL database: NeDB or MongoDB. NeDB is suitable for small-scale databse needs (up to a couple of thousand queries per second). For large-scale deployments, use MongoDB.

Want to hear the good news? NeDB uses a subset of MongoDB's API, so you can code for NeDB and if you need to scale up, just flick a switch and you're using MongoDB.
